# Квест на 8 Марта (March 8th Quest) 🌸

Это проект небольшого поздравительного веб-приложения, разработанного на React + Vite, со скрытой частью для проведения офлайн-квеста по QR-кодам. Каждая команда получает 3 задания: разгадать ребус, расшифровать бинарный код и исправить код на JavaScript.

## 🚀 Настройка и Запуск локально
1. Установите зависимости: `npm install`
2. Запустите dev-сервер: `npm run dev`
3. Откройте `http://localhost:5173/`

## 🌍 Деплой на GitHub Pages
Проект уже настроен для работы на GitHub Pages (используется `HashRouter`, чтобы сканирование QR-кодов и прямые ссылки не приводили к ошибке `404`).

### Инструкция по публикации:
1. Откройте файл `package.json`.
2. Найдите поле `"homepage"`. Оно сейчас выглядит так: `"homepage": "https://<USERNAME>.github.io/<REPO_NAME>",`. Замените `<USERNAME>` на ваш логин GitHub, а `<REPO_NAME>` на название вашего репозитория (например, `8mart`).
3. Откройте конфигурацию `vite.config.js`. Убедитесь, что поле `base: '/8mart/'` (поменяйте `8mart` на точное имя вашего репозитория).
4. Выполните команду деплоя: 
   ```bash
   npm run deploy
   ```
5. Через пару минут ваш квест будет доступен онлайн. Не забудьте перепроверить ссылки в QR-кодах: они должны вести на формат `вашассылка.io/8mart/#/rose-7x2-petal`.

---
**Важно**: Чтобы изменить ребусы, замените `rebusImage` ссылки в `src/data/tasks.js` на реальные URL ваших картинок. Картинки можно загрузить на любой хостинг или поместить в папку `public/assets/` и ссылаться на них как `/assets/my-rebus.png`.
